# IFRS 9 Credit Loss Provisioning Model

## Overview

This repository contains a reference IFRS 9 provisioning data model supporting:

- Stage 1 / 2 / 3 classification
- PD / LGD / EAD calculations
- PIT vs TTC parameter versioning
- Provision auditability
- Macroeconomic overlays
- Expected Credit Loss (ECL) calculations
